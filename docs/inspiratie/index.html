HOW TO BUILD BETTER.

  1. IF POSSIBLE, DON'T BUILD AT ALL
  2. KNOW WHAT'S IMPORTANT
  3. REDUCE INTERVENTIONS, ONE ACTION CAN COMPLY WITH MULTIPLE NEEDS
  4. REDUCE TIME-CONSUMING PROCESSES
  5. REDUCE MATERIALS, THEY WILL BE PRICELESS ONE DAY
  6. RE-USE WHAT IS VALUABLE
  7. KEEP IT SIMPLE, DON'T BE LAZY
  8. DO IT YOURSELF ONLY IF YOU'RE ABLE TO
  9. AVOID DOING THINGS THAT CAN NEVER BE UNDONE
  10. DON'T OBSTRUCT FUTURE IMPROVEMENTS
  11. DON'T MAKE A BIG FUZZ ABOUT A LITTLE CRACK
  12. BE PREPARED, ANTICIPATE
  13. BE GENEROUS AND GRATEFUL

This manifesto was inspired by David Fishli & Peter Weiss' work "How to work better". It reflects my perception of building ethics developed from 2002 towards 2022.
